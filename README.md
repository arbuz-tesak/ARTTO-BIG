
Экран «Зареги» (контакты и локации)
Это отдельный блок внизу сайта, который выполняет функцию футера для регистрации и контактов. 
Здесь есть локации, отправить письмо и другие контакты как телефон и соцсети. 
Цель блока дать посетителю все необходимые данные для связи с приютом.
Презентация:
Экран «Зареги» показывает адреса приютов, контакты и форму для связи. 
Пользователь может сразу написать письмо или узнать, где находится ближайший филиал
